About Car Mileage Predictor - India

Welcome to the Car Mileage Predictor - India, an advanced machine learning application that estimates vehicle fuel efficiency based on technical specifications. This project demonstrates the complete data science pipeline from data collection to model deployment, optimized for Indian roads and fuel prices.

📊 Dataset & Model Performance

Our model is trained on a comprehensive dataset containing over 45,000 vehicles from the U.S. Department of Energy's fuel economy database, plus 31 popular Indian vehicle models. The dataset spans from 1984 to 2026 and includes detailed technical specifications, with special focus on Indian market vehicles.

📈 Model Insights

The Random Forest model identifies several key factors that influence fuel efficiency:

  • Engine Displacement: Larger engines typically consume more fuel
  • Number of Cylinders: More cylinders usually mean lower efficiency
  • Fuel Type: Different fuels have varying energy densities
  • Drive Type: AWD systems add weight and complexity
  • Vehicle Age: Newer technologies improve efficiency

This project demonstrates the practical application of machine learning in solving real-world problems. By combining data science techniques with modern web development, we create tools that provide valuable insights to users.
